# Dash-VBP-Agroneg-cios-2024
Dash de demonstração de conhecimentos em Python para construção de analises estatísticas e apresentação de dados

Para executar de forma local:

Instale as bibliotecas necessárias: Abra o terminal e digite:
pip install dash plotly pandas

Após instalação da biblioteca, execute o dashboard:
python dashboard_vbp.py

Certifique-se de que o arquivo "VBP_Tabela.csv" esteja na mesma pasta que o código python.
